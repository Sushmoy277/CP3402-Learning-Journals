## Content Insights

This week helped me understand how a CMS like WordPress runs in a local development environment before being deployed to a live server. Using XAMPP showed how different components work together to run a dynamic website. Apache acts as the web server, MySQL stores the data, and PHP allows WordPress to communicate with the database.  

Another important insight was understanding the role of the database in a CMS. Unlike static websites where content is written directly into HTML files, WordPress stores most of its content such as posts, pages, and settings inside the MySQL database. The website retrieves this data dynamically when a page is loaded.  

During the process I also learned how configuration files such as `wp-config.php` are used to connect WordPress to the database. If the database name, username, or password are incorrect, WordPress cannot connect and the site will show a database connection error. This helped me understand how sensitive configuration settings are when setting up web applications.  

Overall this practical improved my understanding of how CMS platforms operate behind the scenes and how local environments are used for development and testing before deploying websites online.  



## Career / Employability / Learning Insights

This practical provided useful insight into how developers normally build and test websites before publishing them. Working in a local development environment allows developers to experiment, test features, and fix problems without affecting the live website. This is an important practice in professional web development because it reduces risk and allows changes to be tested safely.  

Another learning insight from this week was recognising the importance of understanding the underlying technologies behind a CMS. While WordPress makes website creation easier through its interface, understanding components like servers, databases, and configuration files helps developers troubleshoot problems more effectively.  

From a learning perspective, combining documentation with practical experimentation helped reinforce the concepts more clearly. Reading about how XAMPP works gave a theoretical understanding, but actually installing WordPress locally and connecting it to the database made the process easier to understand. This approach of combining reading with hands-on practice helped build a clearer understanding of how web systems operate.
