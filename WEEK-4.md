## Content Insights

This week helped me understand how a CMS like WordPress runs in a local development environment before going live on the internet.

Using XAMPP helped me understand how different parts work together:
- Apache works as the web server  
- MySQL stores all the data  
- PHP connects WordPress with the database  

I also learned that WordPress is a dynamic website. Instead of storing content in HTML files, it stores everything in the database and shows it when needed.

From the videos and websites, I learned:
- How to start Apache and MySQL using XAMPP  
- How to create a database using phpMyAdmin  
- How to connect WordPress with the database  
- What localhost means and how to run a site locally  
- What causes database connection errors and how to fix them  
- Basic idea of how WordPress folders and files are structured  

I also understood how the `wp-config.php` file is used to connect WordPress to the database. If the database details are wrong, the website will not work.

Overall, this practical helped me understand how websites work behind the scenes.
