# Week 7 

## Learning Activities & Resources

This week I created a small PHP website based on my UniPath AI idea to show different PHP features. I made  pages (`index.php`, `about.php`) and used `header.php` and `footer.php` with include so that the layout stays the same on both pages.

I used echo to display content, if/else to check conditions, loops like for, while, and foreach, and also created a function with parameters to show a personalised recommendation.

I also used ChatGPT in a limited way to support my learning. I mainly used it to understand concepts step-by-step and fix small errors when I got stuck. Some of the basic prompts I used were:

- "how to run php on localhost mac"
- "example of if else in php simple"
- "php for loop example"
- "how to use include in php"
- "php function with parameters example"

## Content Insights

This week helped me understand how PHP works with HTML to create dynamic websites. I learned that PHP runs in the background and then shows the result in the browser.

Using include for the header and footer helped me understand how to keep the code clean and avoid repeating the same parts again and again.

I also learned how if/else can be used to show different results based on conditions, like checking a student’s budget. This is useful for making websites more personalised.

Loops like for, while, and foreach helped me display repeated content easily instead of writing the same code multiple times. The foreach loop was especially useful for showing lists like countries.

Creating a function helped me understand how to reuse code and keep everything more organised.

## Career/Employability/Learning Insights

This week helped me understand how backend coding works and why it is important in real websites. PHP allows websites to be more interactive and not just static.

I realised that writing clean and organised code is very important in the industry. Using functions and includes makes the code easier to manage and understand.

I also learned that logic like conditions and loops is very important for creating personalised experiences for users. For example, showing different options based on a student’s budget is similar to how real systems work.

I faced a problem while trying to run my PHP files on localhost. At first, when I tried to start the server, it showed an error saying that the PHP command was not found. I realised that PHP was not properly installed on my system. To solve this, I installed PHP using Homebrew and then checked it using the php -v command to make sure it was working. After that, I was able to run the server using php -S localhost:8000 and open my website in the browser. This helped me understand how to set up a local development environment properly and troubleshoot basic setup issues

From this week, I understood that I learn better by doing the work myself instead of only reading. Writing and testing the code helped me understand everything more clearly.

Overall, this week improved my understanding of PHP and helped me learn how real websites work, which will be useful for my future career in IT.
