## Content Insights
This week helped me understand the basic difference between **HTML and CSS**. HTML is used to create the structure of a website, such as headings, text, links, and forms. CSS is used to control how the website looks, including colours, spacing, and layout.

From the YouTube videos about CMS, I learned that a **Content Management System** allows users to create and manage website content using an admin dashboard instead of writing code every time. CMS platforms make it easier to update content, manage menus, and control the design using templates.

I also learned that common CMS platforms include **Joomla, WordPress, and Drupal**, and they are widely used for business and organisational websites. CMS separates content from design, which makes websites easier to maintain and update.

Using Bootstrap helped me understand how responsive layouts can be created more easily. I also learned that a static website is still useful for presenting ideas clearly, even without backend functionality.
